# Amazon-Sales-Dashboard
This is dashboard shows the sales performance of Amazon.

# Project Overview:
    Amazon Stores Sales Dashboard built using Power BI! This dashboard provides a detailed view of the performance of a Amazon Store across multiple dimensions including region, segment, payment mode, shipping mode, and product categories.

# Data Used : 
       Kaggle dataset    

# 🔑KPI Overviews : Total Sales , Total Profit ,Total Orders , Total Products

# 💡Key Insights:
 
       📍Total Sales: 1.57M
   
       💰Total Profit: 0.18M

       📦Total Orders: 5901

       🛍️Total Products: 5901

       🌏Top Region by Sales: West (33%)

       📈Top Category: Office Supplies (0.64M Sales)

       🛥️Most Used Shipping Mode: Standard Class (0.91M Sales)

       🗓️Monthly YOY Analysis of both Profit and Sales for 2019 & 2020

       🗺️State-wise Sales & Profit Visualization with interactive mapping

# This dashboard is designed to help businesses identify trends, track performance, and make data-driven decisions in real time.

# 🔧Tools Used:

 # 📊Techniques:
